# API Routes

This directory contains Next.js API routes organized by function:

## Structure

- `/agents` - Agent endpoints for AI-powered study generation
  - Clinical Protocol Agent
  - Consent & Compliance Agent
  - Enrollment Agent
  - Patient Communication Agent

- `/webhooks` - External integration endpoints
  - Simulated lab results webhook

- `/auth` - Authentication endpoints (Supabase Auth)

- `/studies` - Study CRUD operations

- `/participants` - Participant management

- `/pros` - PRO survey submission and retrieval
